---

## 제품 목표

웹 브라우저에서 마인드맵을 작성하고,  
AI를 이용해 자동 생성 및 확장하며,  
작성된 맵을 Markdown 및 Standalone HTML 형태로 Export/Publish할 수 있는 서비스.

> "마인드맵 작성 → AI 확장 → 웹 퍼블리싱" 전 과정을 하나의 툴에서 처리한다.

---

## 핵심 가치 제안

1. **웹 기반 Mindmap Editor** — 설치 없이 브라우저에서 즉시 사용
2. **노드 단위 레이아웃** — 노드마다 map / graphic / tree 레이아웃을 독립 설정 가능
3. **AI 기반 자동 생성** — 질문 하나로 마인드맵 초안 생성
4. **실시간 자동 저장** — 편집 중 항상 DB에 보존
5. **Standalone HTML Export** — 단일 HTML 파일로 웹서버에 그대로 퍼블리싱
6. **향후 실시간 협업** — 팀 동시 편집 지원 (Phase 3)
