### 1. 기능 목적

* 에디터 주요 기능을 **키보드 단축키로 빠르게 실행**하는 기능
* 마우스 없이도 노드 생성·편집·탐색이 가능한 생산성 UX 제공
* 단축키 도움말 팝업으로 전체 단축키 목록 확인

### 3. 전체 단축키 목록

#### 3.1 노드 편집 (KS-01)

| 단축키                    | 기능                    |
| ---------------------- | --------------------- |
| `Space`                | 자식 노드 추가              |
| `Shift+Space`          | 다음 형제 노드 추가           |
| `Shift+Ctrl+Space`     | 이전 형제 노드 추가           |
| `Ctrl+Space`           | 다중 자식 노드 추가 (bulk)    |
| `Enter` (편집 중)         | 줄바꿈                   |
| `ESC`                  | 편집 종료 / 선택 해제         |
| `Delete` / `Backspace` | 선택 노드 삭제              |
| `Ctrl+D`               | 노드 복제                 |
| `Ctrl+C`               | 복사                    |
| `Ctrl+V`               | 붙여넣기                  |
| `F2` / 더블클릭            | 텍스트 편집 모드 진입          |
| `Tab`                  | 자식 노드 추가 (편집 종료 후)    |

#### 3.2 선택 (KS-02)

| 단축키             | 기능            |
| --------------- | ------------- |
| 클릭              | 단일 선택         |
| `Ctrl+클릭`       | 다중 선택 토글      |
| `Shift+클릭`      | 범위 선택         |
| `Ctrl+A`        | 전체 선택         |
| `ESC`           | 선택 해제         |
| `↑↓←→`          | 인접 노드로 선택 이동  |
| `Ctrl+Shift+A`  | Subtree 전체 선택 |

#### 3.3 캔버스 조작 (KS-03)

| 단축키              | 기능              |
| ---------------- | --------------- |
| `Ctrl+=`         | Zoom In         |
| `Ctrl+-`         | Zoom Out        |
| `Ctrl+0`         | 100% View       |
| `Ctrl+Shift+F`   | Fit Screen      |
| `Ctrl+Enter`     | Center Node     |
| `Alt+F`          | Focus Node View |
| `Space+드래그`      | Pan Canvas      |
| `H`              | Pan 모드 전환       |
| `F11`            | Fullscreen      |
| `ESC`            | Fullscreen 종료   |

#### 3.4 히스토리 (KS-04)

| 단축키                       | 기능   |
| ------------------------- | ---- |
| `Ctrl+Z` / `Cmd+Z`        | Undo |
| `Ctrl+Y` / `Ctrl+Shift+Z` | Redo |

#### 3.5 검색 / 기타 (KS-05)

| 단축키              | 기능          |
| ---------------- | ----------- |
| `Ctrl+F`         | 검색 패널 열기    |
| `?` / `Ctrl+/`   | 단축키 도움말 팝업  |

---

### 4. Kanban 레이아웃 단축키 특수 규칙 (KS-06)

| 단축키                | 일반 레이아웃  | Kanban                                              |
| ------------------ | --------- | --------------------------------------------------- |
| `Space`            | 자식 노드 추가  | depth 0 → 컬럼 추가 / depth 1 → 카드 추가 / depth 2 → ❌ 차단 |
| `Shift+Space`      | 다음 형제 추가  | depth 1 → 다음 컬럼 / depth 2 → 같은 컬럼 내 다음 카드 추가        |
| `Shift+Ctrl+Space` | 이전 형제 추가  | 동일 규칙                                               |

---

### 5. 단축키 도움말 팝업 (KS-07)

```text
┌─────────────────────────────────────────────────┐
│  단축키 도움말                          [✕]      │
├─────────────────────┬───────────────────────────┤
│  노드 편집            │  캔버스                    │
│  Space    자식 추가   │  Ctrl+=   확대             │
│  Delete   삭제       │  Ctrl+-   축소             │
│  Ctrl+Z   Undo      │  Ctrl+Shift+F  전체 맞춤    │
│  Ctrl+Y   Redo      │  Ctrl+Enter  선택 노드 중앙  │
│  Ctrl+A   전체 선택   │  Alt+F   Focus 뷰          │
└─────────────────────┴───────────────────────────┘
```

---

### 6. 규칙

* 텍스트 편집 모드에서는 캔버스 단축키 비활성 (입력 충돌 방지)
* `ESC` 우선순위: 편집 종료 > 검색 닫기 > 선택 해제 > Fullscreen 종료
* Kanban depth 제한 위반하는 단축키는 차단
