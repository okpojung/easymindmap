## 2. 기능별 상세 정의

### CANVAS-01 Zoom In — 확대

| 항목 | 내용 |
|------|------|
| 설명 | 캔버스를 현재 배율에서 단계적으로 확대 |
| 툴바 | `+` 아이콘 클릭 (1단계씩 확대) |
| 단축키 | `Ctrl + =` 또는 `Ctrl + +` |
| 마우스 | `Ctrl + 마우스 휠 위` |
| 확대 기준점 | 마우스 포인터 위치 기준 (휠 조작 시) / 화면 중앙 기준 (버튼/단축키 조작 시) |
| 최대 배율 | 400% (초과 시 동작 없음) |

---

### CANVAS-02 Zoom Out — 축소

| 항목 | 내용 |
|------|------|
| 설명 | 캔버스를 현재 배율에서 단계적으로 축소 |
| 툴바 | `−` 아이콘 클릭 (1단계씩 축소) |
| 단축키 | `Ctrl + -` |
| 마우스 | `Ctrl + 마우스 휠 아래` |
| 축소 기준점 | 마우스 포인터 위치 기준 (휠 조작 시) / 화면 중앙 기준 (버튼/단축키 조작 시) |
| 최소 배율 | 10% (미만 시 동작 없음) |

---

### CANVAS-03 Fit Screen — 전체 화면 맞춤

| 항목 | 내용 |
|------|------|
| 설명 | 맵의 전체 노드가 화면 안에 들어오도록 배율과 위치를 자동 조정 |
| 툴바 | `[ ]` 아이콘 클릭 |
| 단축키 | `Ctrl + Shift + F` |
| 동작 | 전체 노드의 bounding box를 계산 → 화면 크기에 맞게 zoom + pan 동시 적용 |
| 여백 | 화면 가장자리에서 24px 패딩 유지 |

---

### CANVAS-04 Pan Canvas — 캔버스 이동

| 항목 | 내용 |
|------|------|
| 설명 | 맵 전체를 화면에서 자유롭게 이동하는 모드 |
| 툴바 | `✋` 아이콘 클릭 → 손바닥(grab) 커서로 전환 |
| 단축키 (모드 전환) | `H` 키 → Pan 모드 ON / 다른 키 입력 시 이전 모드 복귀 |
| 마우스 (즉시 Pan) | `Space + 드래그` — Pan 모드 전환 없이 일시적 이동 |
| 마우스 (우클릭) | 빈 캔버스 영역 `우클릭 + 드래그` → 손바닥 커서로 Pan |
| 마우스 (미들버튼) | `미들 버튼 + 드래그` → Pan |
| 커서 상태 | Pan 모드 진입: `grab (✋)` → 드래그 중: `grabbing (✊)` |
| Pan 모드 종료 | 툴바 `✋` 재클릭 또는 `Escape` 키 |

> **설계 노트**: 우클릭 단독 클릭은 컨텍스트 메뉴를 여는 용도이므로,  
> 우클릭 + 드래그(마우스 이동 감지)일 때만 Pan으로 동작하고  
> 우클릭 후 이동 없이 놓으면 컨텍스트 메뉴를 표시한다.

---

### CANVAS-05 Center Node — 선택 노드 화면 중앙 이동

| 항목 | 내용 |
|------|------|
| 설명 | 선택된 노드를 화면 정중앙으로 부드럽게 이동 (Pan만 수행, **줌 배율 변경 없음**) |
| 툴바 | `⊙` 아이콘 클릭 |
| 컨텍스트 메뉴 | 노드 우클릭 → `"화면 중앙으로"` 메뉴 항목 |
| 단축키 | `Ctrl + Enter` (노드 선택 상태에서) |
| 애니메이션 | smooth scroll (200ms ease-out) |
| 선택 없을 때 | 아이콘/메뉴 비활성화 (disabled) |

> **100%와의 관계 명확화**  
> - CANVAS-05는 **현재 줌 배율을 그대로 유지**하면서 pan만 수행한다.  
> - 노드를 중앙으로 이동하면서 100%로 보고 싶을 경우:  
>   `CANVAS-05 (Center Node)` + `CANVAS-06 (100% View)` 를 순서대로 실행하거나,  
>   컨텍스트 메뉴에 `"100%로 중앙 이동"` 옵션을 별도 제공하는 방식으로 처리한다.

---

### CANVAS-06 100% View — 100% 보기 ⭐ 신규

| 항목 | 내용 |
|------|------|
| 설명 | 줌 배율을 100% 원본 크기로 즉시 초기화 (위치는 변경하지 않음) |
| 툴바 | `🔍100` 아이콘 클릭 |
| 단축키 | `Ctrl + 0` |
| 동작 | 현재 화면 중심 기준으로 배율만 100%로 변경 |
| 줌 표시 | 툴바 또는 하단 상태바에 현재 배율 `%` 수치 표시 |

---

### CANVAS-07 Fullscreen Mode — 전체화면 모드 ⭐ 신규

| 항목 | 내용 |
|------|------|
| 설명 | 브라우저 UI(탭바, 주소창 등)를 숨기고 맵을 전체화면으로 표시 |
| 툴바 | `⛶` (확장 화살표) 아이콘 클릭 |
| 단축키 (진입) | `F11` 또는 `F` |
| 단축키 (종료) | `ESC` 또는 `F11` |
| 종료 버튼 | 전체화면 진입 시 화면 우상단에 `✕` 버튼 표시 (마우스 상단 호버 시 나타남) |
| 구현 방법 | Web Fullscreen API (`document.documentElement.requestFullscreen()`) |
| 상태 보존 | 전체화면 종료 시 이전 줌 배율 및 pan 위치 그대로 복원 |

---

### CANVAS-08 Focus Node View — 선택 노드 보기 ⭐ 신규

| 항목 | 내용 |
|------|------|
| 설명 | 선택한 노드를 기준으로 **상위 노드를 숨기고** 선택 노드와 하위 노드만 화면에 표시 |
| 툴바 | `📍` 아이콘 클릭 |
| 컨텍스트 메뉴 | 노드 우클릭 → `"이 노드만 보기"` |
| 단축키 | `Alt + F` (노드 선택 상태에서) |
| 진입 표시 | 상단 breadcrumb 또는 배너로 `"[노드명] 포커스 보기 중 — 전체 보기로 돌아가기 ↩"` 표시 |
| 종료 방법 | breadcrumb 클릭 / `ESC` / 툴바 `📍` 재클릭 |
| 동작 상세 | ① 선택 노드 + 하위 노드만 렌더링 (상위 노드 DOM에서 숨김 처리) <br> ② Fit Screen 자동 적용 (선택 서브트리 기준) <br> ③ 원본 맵 데이터는 변경하지 않음 (view 필터만 적용) |
| 선택 없을 때 | 아이콘/메뉴 비활성화 (disabled) |

## 3. 노드 우클릭 컨텍스트 메뉴 — CANVAS 관련 항목

```
노드 우클릭 시 컨텍스트 메뉴
┌─────────────────────────┐
│ ✏️  텍스트 편집          │
│ ➕  자식 노드 추가       │
│ 🗑️  노드 삭제            │
├─────────────────────────┤
│ ⊙   화면 중앙으로        │  ← CANVAS-05
│ 📍  이 노드만 보기       │  ← CANVAS-08
└─────────────────────────┘
```

빈 캔버스 우클릭 시 컨텍스트 메뉴:
```
┌─────────────────────────┐
│ [ ]  전체 화면 맞춤      │  ← CANVAS-03
│ 🔍   100% 보기           │  ← CANVAS-06
└─────────────────────────┘
```

## 5. 줌 배율 단계 정의

```
최소: 10%
단계: 10% → 25% → 50% → 75% → 100% → 125% → 150% → 200% → 300% → 400%
기본: 100%
최대: 400%
```

Zoom In/Out 버튼 클릭 또는 Ctrl+=/- 는 위 단계 기준으로 1단계씩 이동.  
Ctrl + 마우스 휠은 연속 배율 변경 (단계 미적용, 부드럽게).

## 6. 툴바 UI 배치 (참고)

```
캔버스 우하단 툴바 (플로팅):
┌──────────────────────────────────┐
│  +  │  −  │ [] │ ✋ │ ⊙ │ 🔍100 │ ⛶ │ 📍 │
└──────────────────────────────────┘
         ↑ 줌 배율 표시: 100%
```

또는 줌 버튼 사이에 현재 배율 수치를 클릭 가능하게 표시:  
`[ + ]  75%  [ − ]`  → 수치 클릭 시 100%로 즉시 복귀 (CANVAS-06 단축)
