# 쇼핑몰 구축 프로젝트

## 1단계 · 기획

> 기간: 2주 / 산출물: 요구사항 정의서, IA

### 요구사항 정의

- 이해관계자 인터뷰
- 기능 목록 확정

### 정보 구조 설계

- 사이트맵
- 화면 흐름도

## 2단계 · 디자인

> 기간: 3주 / 산출물: 디자인 시스템, 화면 시안

### 디자인 시스템

- 컬러/타이포 토큰
- 컴포넌트 라이브러리

## 3단계 · 개발

> 기간: 6주 / 산출물: 베타 사이트

### 프론트엔드

- 상품 목록/상세
- 장바구니/결제 흐름

### 백엔드

- 상품/주문 API
- 결제 연동
