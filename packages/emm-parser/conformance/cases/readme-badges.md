# AwesomeLib

A tiny library for **fast** parsing.

## About

The lib is described at [docs](https://docs.example.com).
Badge: [![CI](https://img.shields.io/badge/ci-pass-green)](https://ci.example.com/awesomelib)

## Installation

Install via npm:

```bash
npm install awesomelib
```

## Usage

1. Import the library
   `import awesome from 'awesomelib'`
2. Call parse()
3. Profit

- Supports *streaming*
  with backpressure
- Zero dependencies

## API

| Method | Args | Returns |
|--------|------|---------|
| parse | string | AST |
| stringify | AST | string |

> Note: see the [API reference](https://docs.example.com/api) for details.
